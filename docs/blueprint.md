# **App Name**: Badman Tips

## Core Features:

- Today's Tips: Display predictions for the current day.
- 30-Day Tips: Show predictions for the next 30 days in a calendar or list format.
- Tip Quality Assessment: Employ an AI tool that filters existing prediction information by checking the content's length, missing values, and missing keys, to ensure information reliability.

## Style Guidelines:

- Primary color: Dark blue (#243A73) to convey trust and authority.
- Background color: Very light blue (#E6E9F2), near white.
- Accent color: Orange (#D97706) to highlight key tips and calls to action.
- Body and headline font: 'Inter', a clean, modern sans-serif.
- Use simple, outline-style icons to represent different prediction categories.
- Minimalist layout with clear sections for today's tips and the 30-day view.
- Subtle transitions when navigating between different prediction views.
